------------------------------------------------------------------------
This program can:
- find and print out the value of export for any country
- print out the list of exporters for any country that exports 2 of the products that you can use an input
- find the number of exporters for any particular product
- find which countries export exceeds the given value of export as an input and print them out
------------------------------------------------------------------------

PROJECT TITLE: ParsingExportData
PURPOSE OF PROJECT: This program is able to parse through a csv file with information on the countries and find some information about export
VERSION or DATE: 1.0.0 (September 5, 2019)
AUTHOR: Konstantin Krumin
USER INSTRUCTIONS: You can use csv files from the "data" folders and BlueJ environment to test the code
